=========================
Using PvSerialPortIPEngine sample
=========================

This sample shows how to use the PvSerialPortIPEngine class for serial communication
through a GigE Vision IP Engine.

1. Introduction

Using the PvSerialPortIPEngine class, this sample shows how to:
 * Get connect to a device
 * Configure and open a serial port a on a GigE Vision IP Engine
 * Send and receive data through the serial port of the IP Engine

2. Pre-conditions

This samples assumes that:
 * You have a GigE Vision IP Engine connected to a network adapter

3. Description

3.1 PvSerialPortIPEngineSample.cpp

Shows how to use the PvSerialPortIPEngine class for serial communication through a GigE Vision IP Engine. Please refer to the comments in the source code for information on the methods used in this sample.
